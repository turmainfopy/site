Criar script que insere sumário em um documento HTML

  • O script deve ser feito em um arquivo .js que será inserido no arquivo HTML pela tag "Script";

  • Cada elemento da lista deve ser um link para a seção "h2" que ele representa;
  • Logo depois de cada "h2" insira um link para o índice (lista);
  • Todos elementos da lista e todos os links devem ser criados em JavaScript
